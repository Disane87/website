支持所有主流的 npm 客户端

npm, yarn 和pnpm是任何开发流程的一部分，我们试图在最新版本中赶上。

使 DevOps 工作轻松

我们有官方的 Docker 镜像可供使用 

docker pull verdaccio/verdaccio

和 ** Kubernetes Helm** 支持, 便于部署

helm repo add verdaccio https://charts.verdaccio.org
helm repo update
helm install --name npm verdaccio/verdaccio

插件支持

Verdaccio 是基于插件的，支持身份验证、中间件和存储插件。只需选择一个现有的即可使用或开发一个属于您自己的。
